UX/UI Design

Web Developement

Why was this Website made?

Web development in the context of a lecture with the use of HTML and CSS and without tools such as Bootstrap. The task was to create a website consisting of several pages and a responsive design, so that it is usable for computers as well as tablets and smartphones and changes flexibly with the page width. The website contains linking and various interactive elements such as drop-down menus and hover-over effects

Role: UX & UI Designer
Tools: Figma, Marvel App
Timeline: 6 weeks

Design Process Overview

Prototyping

After the requirements for the website were clarified, wireframes were created as a basis for development. Here, special emphasis was placed on drawing the containers and boxes that will later be needed in HTML and CSS.
